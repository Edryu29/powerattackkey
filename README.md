Use this key to trigger right hand power attacks:

> iDualWieldKeyAlt1 = -1    ;Default is -1

> iComboKey = 45            ;Default is -1

> iComboKeyAlt1 = 281       ;Default is -1

> iComboKeyAlt2 = -1        ;Default is -1
